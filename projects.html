Graphics Rendering Engine

The program uses the Vulkan API to render a rectangle with an applied texture. The program currently initializes the API and only implements the required functions to perform the render.

This application is currently in development and its purpose is for me to learn graphics API and the graphics pipeline. Vulkan is used for its cross-compatibility and feature support/control. This project was to learn more about the rendering pipeline and become more familiar with low-level optimization and graphics hardware.
